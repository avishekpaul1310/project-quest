# Project Quest: Project Management Learning Game

A gamified learning platform for Project Management concepts built with Django.

## Overview

Project Quest helps users learn Project Management concepts through interactive missions and quizzes. The game presents PM concepts in an engaging way, with missions covering fundamental topics like Project Charter, Stakeholder Management, and Risk Management.

## Features (Phase 1 MVP)

- Sequential learning missions (3 initial missions)
- Educational content with Key Concepts and Best Practices
- Multiple-choice questions (5 per mission)
- Basic progress tracking
- Simple scoring system (10 points per correct answer)

## Technology Stack

- Python 3.8+
- Django 4.2+
- SQLite (for Phase 1)
- Bootstrap 5 (for basic styling)
- django-crispy-forms
